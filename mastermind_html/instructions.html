    How to Play Mastermind

    Solve the hidden code in under 10 guesses and claim your title of Mastermind!
    You, the Codebreaker will place four code pegs into the first row of holes, attempting to duplicate the exact colors and positions of the secret code.
    Then, red and white key pegs will be placed next to your guess as clues to guide you toward the secret code.

    1. A red key peg for each code peg that's the right color and in the right position
    2. A white key peg for each code peg that's the right color, but not in the right position
    3. No key pegs indicate a color is not used in the code
    The key pegs will not be given in any particular order. It's part of the challenge for the Codebreaker to figure out which key pegs correspond to which code pegs.
    The Codebreaker should remember that one key peg corresponds to one code peg, and a red key peg takes precedence over a white one.
