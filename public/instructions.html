    How to Play Mastermind

    Solve the hidden code in under 10 guesses and claim the title of Mastermind!

    You, the Codebreaker will place four code pegs into the bottom row of holes, attempting to duplicate the exact colors and positions of a secret, 4-color code.

    Then, red and white key pegs will be placed to the right of your guess as clues to guide you toward the secret code.

    • A red key peg corresponds to a code peg that's the correct color and in the correct position

    • A white key peg corresponds to a code peg that's the correct color, but not in the correct position

    • An empty key peg slot indicates that a color is not used in the code

    The key pegs will not be given in any particular order. It's part of the challenge for the Codebreaker to figure out which key pegs correspond to which code pegs.

    The Codebreaker should remember that one key peg corresponds to one code peg, and a red key peg takes precedence over a white one.

    Your next guess will be placed in the line above your first guess, and you will continue moving up the board until the hidden code is revealed.
